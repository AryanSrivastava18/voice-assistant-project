# voice-assistant-project
A smart AI-based Voice Assistant project built to perform tasks using voice commands. It can recognize speech, respond intelligently, and automate basic operations like searching, opening apps, and providing information. Developed using Python with speech recognition and text-to-speech integration.
AI Voice assistant 

This project is an AI-based Voice Assistant developed using Python. It can understand voice commands, process them, and respond intelligently. The assistant can perform tasks like searching the web, opening applications, telling time/date, and providing information using speech recognition and text-to-speech technologies.

Features:
Voice command recognition
 Text-to-speech response
 Web search automation
 Open applications
 Time & date information
 Wikipedia search
 Play YouTube videos
 Technologies Used
Python
SpeechRecognition
PyAudio
pyttsx3 / gTTS
Wikipedia API
pywhatkit
